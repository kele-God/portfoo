# Kelechi Egbuta — Software Engineer Portfolio

This is a personal portfolio website for Kelechi Egbuta, a software engineer and systems architect. The site showcases professional experience, selected projects, skills, and contact information.

## Features
- **Modern, Responsive Design**: Custom CSS and typography for a unique, professional look.
- **Custom Cursor & Animations**: Enhanced user experience with a gold-themed cursor and smooth reveal animations.
- **Navigation**: Fixed top navigation bar with smooth scrolling and a mobile-friendly drawer menu.
- **Hero Section**: Name, title, and a visual with animated rings and stats.
- **About Section**: Professional summary, philosophy, and a grid of skills (languages, frameworks, DevOps, UI/UX, data/cloud, quality/process).
- **Projects Section**: Five detailed project cards, each with stack, impact, and duration.
- **Methodology Section**: Four core engineering principles (Agile, TDD, CI/CD, UI/UX).
- **Contact Section**: Email (protected), LinkedIn, GitHub, and Resume download.
- **Footer**: Copyright, location, and a builder's note.

## Getting Started (Current Static Version)
1. Open `index.html` in your browser to view the portfolio.
2. All styles and scripts are embedded in the HTML file; no build step is required.

## Roadmap for React Version
1. **Bootstrap with Create React App or Vite**

5. **State Management**: Use React Context or Redux as needed.
